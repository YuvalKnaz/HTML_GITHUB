# Manchester United Mini Site

A simple static website about Manchester United FC with 4 pages and linked navigation.

## Pages
- `index.html` - Club home and overview
- `squad.html` - Current squad highlights
- `history.html` - Club history
- `player.html` - Specific player profile (Marcus Rashford)

## Assets
- `style.css` - shared styling
- `images/` - vector graphics used as page heroes

## How to run
1. Open `index.html` in a browser (double-click or right-click and open).
2. Navigate the pages via the top menu.

## Improvements you can add
- Add more players to `player.html` and a player selection list.
- Include external data or JSON for dynamic content.
- Add media queries and mobile header behavior.
